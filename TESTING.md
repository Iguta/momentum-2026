# Testing Guide

## Commands (run from momentum-2026/codebase)

- `npm test`
- `npm run build`

## Must-Have Coverage (PRD)

- Task Management
  - Covered by `adds a task with description, due date, and priority weight`
- Goal Setting
  - Covered by `creates a goal across yearly/monthly/daily timeframes`
- Calendar View
  - Covered by `renders the calendar view with a zoomable day list`
- Progress Dashboards
  - Covered by `shows progress dashboards for today, week, month, and year`
