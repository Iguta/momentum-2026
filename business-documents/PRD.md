# Product Requirements Document (PRD)

## Project Name  
**momentum-2026**

## Overview  
Build a personal goal + task tracker web app with a modern, beautiful UI/UX and fully responsive design (mobile/tablet/desktop).

## Key Features  
- **Task Management**  
  - Add tasks with description, due date, and priority weight.
  - Edit and delete tasks.
  - View tasks in a list or card format.
  - Prioritize tasks based on user-defined criteria.
  - Search for tasks easily.

- **Goal Setting**  
  - Define new year resolutions, monthly goals, and daily goals.
  - Support themes for goals (e.g., Spiritual, Physical Fitness, Academics, Career).
  - Users can create/manage their own categories for both tasks and goals.

- **Calendar View**  
  - Visualize tasks by date in a calendar format.
  - Expand or zoom into days that contain tasks for better visibility.

- **Progress Dashboards**  
  - Provide charts for tracking progress today, this week, this month, and this year.

## UI/UX Requirements  
- Modern, minimal, and highly polished design.
- Default dark theme with soft accent colors.
- Responsive layout across devices.
- Calendar-first layout with smooth interaction for day expand/zoom.
- Clean task cards, subtle animations, and simple progress charts.
- Focus on clarity, calm, and personal productivity.
