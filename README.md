# momentum-2026

A personal goal + task tracker web app with a modern, polished UI/UX and responsive layout for mobile, tablet, and desktop. The app focuses on calm productivity with a calendar-first layout, task management, goal setting, and progress dashboards.

## Key Features
- Task management with description, due date, priority weight, list/card views, prioritization, and search.
- Goal setting for yearly, monthly, and daily goals with themed categories.
- Calendar view to visualize tasks by date with day expand/zoom interactions.
- Progress dashboards with charts for today, this week, this month, and this year.

## UI/UX
- Modern, minimal design with a default dark theme and soft accent colors.
- Clean task cards, subtle animations, and simple progress charts.
- Calendar-first layout with smooth interactions.
